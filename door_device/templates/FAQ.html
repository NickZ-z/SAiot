Perguntas frequentes

Para a adição de um dispositivo no sistema é necessário, quando o sistema web estiver conectado ao broker 'www.maqiatto.com', o dispositivo deve enviar um JSON padronizado para realizar o registro no BANCO DE DADOS.

Exemplo do padrão JSON:

{ "funcao" : "Porta", "mac":"00:11:22:33:44:55", "status":"aberta" }

Verifique o código do erro.

Veja nessa lista um conjunto de erros possíveis:

  • conflited_mac - Endereço MAC inválido.
  • time_over - O tempo limite de conexão cessou.
  • JSON_invalid - A mensagem JSON recebida está incompleta.
  • Broken_msg - A mensagem recebida não está em JSON.
  • conflited_json - JSON conflituoso, cheque o 'status'.

Na atualidade, o sistema só suporta microcontroladoras com acesso a redes wifi e que são programáveis. Como por exemplo: O Arduino e ESP32.
